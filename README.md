# NDS Chinese Visual Novel Archive

Preservation archive of Chinese visual novels and related VNDS/AVG tools ported to or used on Nintendo DS.

中文视觉小说移植作品及相关 VNDS、AVG Maker 工具的保存归档。现有资料主要整理自互联网，原 README 记录的来源为 `yayabo.com`，Release 中的作品以中文版本为主。

## Archive layout

The repository itself contains this index. Large preservation files are stored as GitHub Release assets, including:

- Nintendo DS visual novel ports;
- VNDS-compatible packages and runtimes;
- AVG Maker DS resources;
- cover images and supporting files.

Some archives are split into numbered parts. Download every part in the same set before extracting it.
